It has come to my attention that it is a cardinal sin to use a website builder as a frontend developer so I set out to make my website as modern as possible while putting my personal design philosphy on display. This website exists on a single webpage and loads information based on user input. This was my attempt to take the design structure of event-driven systems to frontend design.

Since the purpose of a resume website is to create a single static page that displays as much of my abilities as possible while not overwhelming the user (that's you) all content on this page is marked as hidden until the user needs it.

In recent years it's also become standard for websites to be both dynamic and customizable. So this website does include a light mode. However I believe in customization, so if you press the hamburger menu above you'll notice you can apply your own colors to the website.
